# 仕入れ還付管理 操作説明書

この説明書は、仕入れ還付管理アプリを日々の入力・確認・出力に使う人向けの手順です。

> 最終的な税務判断や申告処理は、税理士に確認してください。

## 1. アプリでできること

- 中古品などの仕入明細を登録できます。
- 証憑画像を明細に複数枚添付できます。
- 古物商特例、準古物、インボイス、経過措置の判定を確認できます。
- 控除対象仕入税額を自動計算できます。
- 月別、担当別、支店別、チャネル別に確認できます。
- CSV、Excel、PDF印刷向けに出力できます。
- 証憑画像をZIPでまとめて保存できます。
- バックアップを作成・復元できます。
- 設定すればAI抽出とクラウド同期も使えます。

## 2. 基本画面

画面は大きく分けて次の構成です。

- 仕入登録: 仕入明細を入力・保存する場所
- AI抽出: スクリーンショットや明細テキストから仕入候補を読み取る場所
- 一覧・集計: 登録済みの仕入を確認・検索・集計する場所
- CSV入出力: CSV、Excel、PDF、証憑ZIP、バックアップを扱う場所
- 設定: AI接続、同期、支店・担当などのマスタを設定する場所

画面が狭い場合は、各パネルが縦に並びます。

## 3. 仕入を1件登録する

1. `仕入登録` パネルで仕入日を入力します。
2. 支店、チャネル、担当を選びます。
3. 商品名、品目、数量、金額（税込）、税率を入力します。
4. 古物区分、棚卸資産、適格事業者、取引形態を選びます。
5. 相手方氏名、相手方住所、メモを必要に応じて入力します。
6. 証憑画像がある場合は `証憑画像` から添付します。明細画像、商品ページ画像、現物写真などを複数枚まとめて選択できます。
7. 画面下部の判定結果と注意表示を確認します。
8. `保存` を押します。

保存後、明細は `一覧・集計` に表示されます。

## 4. 判定結果の見方

仕入登録フォームの下に、控除区分と控除対象仕入税額が表示されます。

主な控除区分:

- 古物商特例: 古物商特例として100%控除対象になる可能性があるもの
- 古物商特例（準古物）: 準古物として扱うもの
- 要インボイス: 適格請求書などの保存確認が必要なもの
- 経過措置 80% / 50%: インボイス経過措置の対象になるもの
- 控除不可 / 要確認: 情報不足または控除対象外の可能性があるもの

注意表示が出た場合は、相手方情報や証憑の保存状況を確認してください。

## 5. 証憑画像を扱う

### 登録時に添付する

仕入登録フォームの `証憑画像` から画像を選ぶと、明細と一緒に保存されます。複数枚をまとめて選択できます。

### 登録済み証憑を見る

一覧の `証憑` 欄には代表サムネイルと画像枚数が表示されます。サムネイルを押すと拡大表示できます。

複数枚ある場合は、証憑表示画面で `前へ` / `次へ` またはサムネイルを押して切り替えられます。

### 証憑をダウンロードする

拡大表示画面で `ダウンロード` を押すと、現在表示している証憑画像を保存できます。

### 証憑をまとめて保存する

`CSV入出力` の `証憑ZIP` を押すと、現在の絞り込み条件に合う全証憑画像を1つのZIPファイルで保存できます。1件の仕入に複数枚ある場合も、すべて出力されます。

## 6. 登録済み明細を確認する

`一覧・集計` パネルで登録済みの仕入を確認できます。

できること:

- 月別タブで対象月を切り替える
- 担当、チャネル、支店、控除区分で絞り込む
- 商品名、相手方、日付、金額、メモで検索する
- 日付、金額、控除税額で並び替える
- 明細の編集・削除をする

編集したい場合は、一覧の編集ボタンを押します。フォームに内容が戻るので、修正して再度 `保存` します。

## 7. CSV・Excel・PDFを出力する

`CSV入出力` の書き出しボタンから出力できます。

- 仕入一覧CSV: 通常の仕入明細一覧
- 古物商帳簿CSV: 古物商特例の確認用帳簿
- 索引簿CSV: 電帳法向けの索引情報
- Excel: サマリーと明細を含むExcelファイル
- PDF印刷: ブラウザの印刷画面を開くレポート
- 証憑ZIP: 証憑画像をまとめたZIP
- バックアップ: 明細、証憑、設定を含むJSONバックアップ

出力対象は、現在の月タブやフィルタ条件に合わせた明細です。

## 8. CSVを読み込む

1. `CSV入出力` の読み込み欄でCSVファイルを選びます。
2. 証憑画像も紐づけたい場合は、先に `証憑画像` で画像ファイルを選びます。
3. CSV内の `証憑ファイル名` と画像ファイル名が一致すると、自動で明細に紐づきます。複数ファイル名が ` / ` 区切りで入っている場合は、複数枚を同じ明細に紐づけます。

CSVの形式が分からない場合は、`テンプレート` を押してひな形を保存してください。

## 9. バックアップと復元

### バックアップする

`CSV入出力` の `バックアップ` を押すと、JSONファイルを保存できます。

バックアップに含まれるもの:

- 仕入明細
- 証憑画像
- 支店・担当・チャネルなどのマスタ
- 一部の設定

合言葉などの秘密情報はバックアップに含めません。

### 復元する

`バックアップ復元` からJSONファイルを選ぶと、保存済みのデータを復元できます。

復元は既存データに追加・上書きされます。完全に入れ替えたい場合は、必要なバックアップを保存してから `全削除` を使ってください。

## 10. AI抽出を使う

AI抽出は、Netlify Functions と Anthropic API の設定が済んでいる場合に使えます。

1. 設定画面で `AI接続` を `中継` にします。
2. `中継URL` に `/.netlify/functions/extract` を入力します。
3. 合言葉を設定している場合は、Netlify側と同じ値を入力します。
4. `AI抽出` パネルでスクリーンショットや明細画像を選びます。
5. 必要なら明細テキストも貼り付けます。
6. `AIで読み取る` を押します。
7. 抽出結果を確認・修正します。
8. `まとめて追加` を押して仕入明細として保存します。

AI抽出結果は必ず確認してください。日付、金額、相手方、区分が読み違えられる場合があります。

## 11. クラウド同期を使う

同期は Netlify Blobs を使います。複数端末で同じデータを使いたい場合に設定します。

1. 設定画面で `同期` を `クラウド` にします。
2. `同期URL` に `/.netlify/functions/sync` を入力します。
3. 合言葉を設定している場合は、Netlify側と同じ値を入力します。
4. `保存` を押します。
5. 画面上部の `更新` を押すとクラウド側のデータを取り込みます。

注意:

- 同期は基本的な更新日時と削除マーカーに対応しています。
- 複数端末で同じ明細を同時編集した場合の細かい競合解決画面はまだありません。
- 重要な作業前にはバックアップを取ってください。

## 12. 支店・担当・品目などを変更する

1. 画面上部の `設定` を押します。
2. 支店、チャネル、担当、品目を改行区切りで編集します。
3. `保存` を押します。

保存後、仕入登録フォームやフィルタの選択肢に反映されます。

## 13. よくある困りごと

### AI抽出が使えない

- AI接続が `中継` になっているか確認してください。
- 中継URLが `/.netlify/functions/extract` になっているか確認してください。
- Netlify側に `ANTHROPIC_API_KEY` が設定されているか確認してください。
- 合言葉がNetlify側の `SHARED_SECRET` または `SYNC_SECRET` と一致しているか確認してください。

### 同期でエラーになる

- 同期が `クラウド` になっているか確認してください。
- 同期URLが `/.netlify/functions/sync` になっているか確認してください。
- 合言葉がNetlify側と一致しているか確認してください。
- Netlifyのデプロイが成功しているか確認してください。

### Excelが出力できない

Excel出力は外部ライブラリの読み込みに依存しています。読み込めない環境ではCSVを書き出してください。

### 証憑ZIPが大きい

証憑ZIPは画像を無圧縮でまとめます。件数が多い場合は、月別やフィルタで対象を絞ってから保存してください。

### 入力欄が押せない、画面が崩れる

ブラウザを更新してください。改善しない場合は、古いデプロイが表示されている可能性があるため、Netlifyの最新デプロイを確認してください。

## 14. 運用のおすすめ

- 仕入入力後は、月ごとにCSVまたはExcelを保存する
- 証憑ZIPを月ごとに保存する
- 大きな修正前にはバックアップを取る
- AI抽出結果は必ず人が確認する
- 税務判断が難しい明細はメモに確認内容を残す
