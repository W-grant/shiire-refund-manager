# 仕入れ還付管理 操作説明書

この説明書は、仕入れ還付管理アプリを日々の入力・確認・出力に使う人向けの手順です。

> 最終的な税務判断や申告処理は、税理士に確認してください。

## 1. アプリでできること

- 中古品などの仕入明細を登録できます。
- 証憑画像を明細に複数枚添付できます。
- 古物商特例、準古物、インボイス、経過措置の判定を確認できます。
- 控除対象仕入税額を自動計算できます。
- 月別、担当別、支店別、チャネル別に確認できます。
- CSV、Excel、PDF印刷向けに出力できます。
- 証憑画像をZIPでまとめて保存できます。
- 月次の税理士提出パッケージをZIPで保存できます。
- バックアップを作成・復元できます。
- 設定すればAI抽出とクラウド同期も使えます。

## 2. 基本画面

画面は大きく分けて次の構成です。

- 仕入登録: 仕入明細を入力・保存する場所
- AI抽出: スクリーンショットや明細テキストから仕入候補を読み取る場所
- 経営ダッシュボード: 売上、粗利、利益率、未処理タスクを確認する場所
- 販売管理: CATAWIKI / EBAY などの販売状況、販売価格、手数料、粗利を管理する場所
- 一覧・集計: 登録済みの仕入を確認・検索・集計する場所
- CSV入出力: CSV、Excel、PDF、証憑ZIP、バックアップを扱う場所
- 設定: AI接続、同期、支店・担当などのマスタを設定する場所

画面が狭い場合は、各パネルが縦に並びます。

## 3. 仕入を1件登録する

1. `仕入登録` パネルで仕入日を入力します。
2. 支店、チャネル、担当を選びます。
3. 商品名、品目、数量、金額（税込）、税率を入力します。
4. 古物区分、棚卸資産、適格事業者、取引形態を選びます。
5. 相手方氏名、相手方住所、メモを必要に応じて入力します。
6. 証憑画像がある場合は `証憑画像` から添付します。明細画像、商品ページ画像、現物写真などを複数枚まとめて選択できます。
7. 画面下部の判定結果と注意表示を確認します。
8. `保存` を押します。

保存後、明細は `一覧・集計` に表示されます。

## 4. 判定結果の見方

仕入登録フォームの下に、控除区分と控除対象仕入税額が表示されます。

主な控除区分:

- 古物商特例: 古物商特例として100%控除対象になる可能性があるもの
- 古物商特例（準古物）: 準古物として扱うもの
- 要インボイス: 適格請求書などの保存確認が必要なもの
- 経過措置 80% / 50%: インボイス経過措置の対象になるもの
- 控除不可 / 要確認: 情報不足または控除対象外の可能性があるもの

注意表示が出た場合は、相手方情報や証憑の保存状況を確認してください。

## 5. 証憑画像を扱う

### 登録時に添付する

仕入登録フォームの `証憑画像` から画像を選ぶと、明細と一緒に保存されます。複数枚をまとめて選択できます。

### 登録済み証憑を見る

一覧の `証憑` 欄には代表サムネイルと画像枚数が表示されます。サムネイルを押すと拡大表示できます。

複数枚ある場合は、証憑表示画面で `前へ` / `次へ` またはサムネイルを押して切り替えられます。

### 証憑をダウンロードする

拡大表示画面で `ダウンロード` を押すと、現在表示している証憑画像を保存できます。

### 証憑をまとめて保存する

`CSV入出力` の `証憑ZIP` を押すと、現在の絞り込み条件に合う全証憑画像を1つのZIPファイルで保存できます。1件の仕入に複数枚ある場合も、すべて出力されます。

## 6. 登録済み明細を確認する

`一覧・集計` パネルで登録済みの仕入を確認できます。

できること:

- 月別タブで対象月を切り替える
- 担当、チャネル、支店、控除区分で絞り込む
- 商品名、相手方、日付、金額、メモで検索する
- 日付、金額、控除税額で並び替える
- 明細の編集・削除をする

編集したい場合は、一覧の編集ボタンを押します。フォームに内容が戻るので、修正して再度 `保存` します。

## 7. CSV・Excel・PDFを出力する

`CSV入出力` の書き出しボタンから出力できます。

- 仕入一覧CSV: 通常の仕入明細一覧
- 古物商帳簿CSV: 古物商特例の確認用帳簿
- 索引簿CSV: 電帳法向けの索引情報
- Excel: サマリーと明細を含むExcelファイル
- PDF印刷: ブラウザの印刷画面を開くレポート
- 証憑ZIP: 証憑画像をまとめたZIP
- 提出ZIP: 税理士提出用の月次パッケージ
- バックアップ: 明細、証憑、設定を含むJSONバックアップ

出力対象は、現在の月タブやフィルタ条件に合わせた明細です。

## 7.5 売上・利益を見る

`経営ダッシュボード` では、販売管理に入力した情報をもとに、月別の売上と粗利を確認できます。

確認できること:

- 対象月ごとの売上
- 対象月ごとの粗利
- 利益率
- 未処理件数
- 担当者別の粗利
- CATAWIKI / EBAY / その他の販売先別粗利
- 証憑なし、利用先未定、未出品、準備中、未売却などの未処理タスク

対象月は `経営ダッシュボード` 右上の `対象月` で切り替えられます。

## 7.6 販売情報を管理する

`販売管理` では、仕入レコードに対して販売情報を登録できます。

主な入力項目:

- 仕入商品
- 販売先: CATAWIKI / EBAY / その他
- 状態: 未出品、準備中、出品中、売却済み、キャンセル、返品、保留
- 出品日、販売日
- 販売価格
- 販売手数料、決済手数料、国内送料、海外送料、その他費用
- 管理番号、SKU、メモ

販売情報を保存すると、粗利と利益率が自動計算され、経営ダッシュボードにも反映されます。

販売一覧は、販売先、状態、担当、商品名・管理番号・SKUで絞り込めます。

## 7.7 Googleスプレッドシートへ送信する

`CSV入出力` の `スプシへ送信` を押すと、アプリ内の仕入データと販売データをGoogleスプレッドシートへ反映します。

反映される主なシート:

- `仕入管理`
- `販売管理`
- `CATAWIKI`
- `EBAY`
- `同期ログ`

`CATAWIKI` と `EBAY` は、利用先が対象サービスまたは `共通` の仕入を抽出した管理用ビューです。

`同期ログ` には、同期日時、結果、件数、エラー内容が残ります。

## 8. 税理士提出パッケージを作る

毎月1回、税理士へ提出するファイル一式をZIPで保存できます。

1. `CSV入出力` の `税理士提出` で対象月を選びます。
2. `提出ZIP` を押します。
3. `税理士提出_YYYY-MM.zip` が保存されます。

ZIPには次のファイルが入ります。

- `01_仕入一覧.csv`
- `02_古物商特例帳簿.csv`
- `03_電帳法索引簿.csv`
- `04_月次サマリー.xlsx`
- `05_月次レポート.pdf`
- `証憑/` フォルダ内の全証憑画像

1件の仕入に複数の証憑画像がある場合も、すべて `証憑/` に入ります。同名ファイルは自動で名前が調整されます。

## 9. CSVを読み込む

1. `CSV入出力` の読み込み欄でCSVファイルを選びます。
2. 証憑画像も紐づけたい場合は、先に `証憑画像` で画像ファイルを選びます。
3. CSV内の `証憑ファイル名` と画像ファイル名が一致すると、自動で明細に紐づきます。複数ファイル名が ` / ` 区切りで入っている場合は、複数枚を同じ明細に紐づけます。

CSVの形式が分からない場合は、`テンプレート` を押してひな形を保存してください。

## 10. バックアップと復元

### バックアップする

`CSV入出力` の `バックアップ` を押すと、JSONファイルを保存できます。

バックアップに含まれるもの:

- 仕入明細
- 証憑画像
- 支店・担当・チャネルなどのマスタ
- 一部の設定

合言葉などの秘密情報はバックアップに含めません。

### 復元する

`バックアップ復元` からJSONファイルを選ぶと、保存済みのデータを復元できます。

復元は既存データに追加・上書きされます。完全に入れ替えたい場合は、必要なバックアップを保存してから `全削除` を使ってください。

## 11. AI抽出を使う

AI抽出は、Cloudflare Pages Function `/extract` と Anthropic API の設定が済んでいる場合に使えます。

1. 設定画面で `AI接続` を `中継` にします。
2. `中継URL` に `/extract` を入力します。
3. 合言葉を設定している場合は、Cloudflare側の `SHARED_SECRET` と同じ値を入力します。
4. `AI抽出` パネルでスクリーンショットや明細画像を選びます。
5. 必要なら明細テキストも貼り付けます。
6. `AIで読み取る` を押します。
7. 抽出結果を確認・修正します。
8. `まとめて追加` を押して仕入明細として保存します。

AI抽出結果は必ず確認してください。日付、金額、相手方、区分が読み違えられる場合があります。

## 12. クラウド保存を使う

現在の本番版はSupabaseに保存します。ログイン後に登録・編集・削除した仕入データ、証憑画像、税理士提出ZIPはクラウド側に保存されます。

1. Supabase Authでログインします。
2. 仕入登録フォームから保存します。
3. 画面上部の `更新` を押すとSupabase側の最新データを再取得します。

注意:

- 未ログインでは業務画面を利用できません。
- Version1では通常運用をSupabase保存に寄せています。
- 重要な作業前にはバックアップを取ってください。

## 13. 支店・担当・品目などを変更する

1. 画面上部の `設定` を押します。
2. 支店、チャネル、担当、品目を改行区切りで編集します。
3. `保存` を押します。

保存後、仕入登録フォームやフィルタの選択肢に反映されます。

## 14. よくある困りごと

### AI抽出が使えない

- AI接続が `中継` になっているか確認してください。
- 中継URLが `/extract` になっているか確認してください。
- Cloudflare Pages側に `ANTHROPIC_API_KEY` が設定されているか確認してください。
- 合言葉を使う場合はCloudflare側の `SHARED_SECRET` と一致しているか確認してください。

### クラウド保存でエラーになる

- ログインしているか確認してください。
- 管理者または保存権限のあるユーザーか確認してください。
- Cloudflare Pagesの最新デプロイが成功しているか確認してください。
- SupabaseのRLS、Storage bucket、環境変数が設定済みか確認してください。

### Excelが出力できない

Excel出力は外部ライブラリの読み込みに依存しています。読み込めない環境ではCSVを書き出してください。

### 証憑ZIPが大きい

証憑ZIPは画像を無圧縮でまとめます。件数が多い場合は、月別やフィルタで対象を絞ってから保存してください。

### 入力欄が押せない、画面が崩れる

ブラウザを更新してください。改善しない場合は、古いデプロイが表示されている可能性があるため、Cloudflare Pagesの最新Productionデプロイを確認してください。

## 15. 運用のおすすめ

- 仕入入力後は、月ごとにCSVまたはExcelを保存する
- 証憑ZIPを月ごとに保存する
- 大きな修正前にはバックアップを取る
- AI抽出結果は必ず人が確認する
- 税務判断が難しい明細はメモに確認内容を残す
